Monitoring system 是一个前端解决方案，基于 vue 2 和 element-ui 2.12.0 开发的后台管理系统。功能实时记录多台设备温度、湿度、烟雾浓度，超过阈值报警提示，管理工单，用户，设备等多种丰富组件。能够快速帮助中小型企业搭建后台产品模型。
